Arduino class for write data to external eeprom in i2c bus (zs-042 module)(ds3231).
	

+ FLOAT -> 4 byte
+ zs-042 => first 16 byte use DS3231

**functions:**

+ float_write
+ float_read
+ float_update
+ is_avaible           -> in i2c address
+ device_scanner -> i2c

**Tested in:**

+ ESP32
+ Arduino NANO
